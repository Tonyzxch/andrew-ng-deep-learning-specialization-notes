# Andrew Ng Deep Learning Specialization Notes

这是我学习 Andrew Ng《Deep Learning Specialization》课程时整理的听课笔记、测验记录与作业实验仓库。

虽然我是结合视频资源学习的，但内容主线对应的是 Coursera 上的 Deep Learning Specialization 课程体系，包括神经网络基础、深度学习优化、机器学习策略、卷积神经网络以及序列模型等核心内容。

本仓库以 Notebook 为主，按照课程与周次顺序整理，包含课程笔记、测验题记录以及部分编程作业实现，适合作为系统复习深度学习经典内容的学习资料。

## 内容概览

本仓库当前主要覆盖以下内容：

- 课程 1：神经网络与深度学习
- 课程 2：改善深层神经网络训练
- 课程 3：机器学习策略
- 课程 4：卷积神经网络
- 课程 5：序列模型

具体内容包括：

- Logistic 回归、浅层神经网络、深层神经网络
- 初始化、正则化、梯度检验、优化算法与 TensorFlow 入门
- 误差分析、数据划分与机器学习策略
- 卷积网络、ResNet、目标检测、人脸识别、风格迁移
- RNN、LSTM、词向量、机器翻译、注意力机制与语音识别

## 使用方式

1. 克隆仓库到本地。
2. 使用 Jupyter Notebook、JupyterLab 或 VS Code 打开 `.ipynb` 文件。
3. 按编号顺序阅读和运行 Notebook。
4. 根据 Notebook 内容自行准备相关依赖、数据集和权重文件。

## 环境建议

- Python 3.x
- NumPy
- TensorFlow / Keras
- Jupyter Notebook / JupyterLab / VS Code Notebook

> 说明：为了控制仓库体积，课程作业中使用到的数据集、音频文件、图片资源、模型文件及部分中间结果未保留在仓库中。如需完整复现，请根据 Notebook 中的实际内容自行下载相关资源，并按本地环境修改对应路径配置。
